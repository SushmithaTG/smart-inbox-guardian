# UI Ideas

## Login Page
- Email input
- Password input
- Login Button
- Register Button
- "Forgot Password?" link     (every real app has this)

## Register Page             
- Full Name input
- Email input
- Password input
- Confirm Password input
- Register Button

## Dashboard
- Total Important Emails     (with a number badge)
- Upcoming Interviews        (with date + time)
- Recent Notifications       (latest 5, with "mark as read")
- Application Status         (Applied/OA/Interview/Offer/Rejected)

## Gmail Connection Page
- Connect Gmail Button
- List of connected accounts (user may connect 2 accounts)
- Disconnect button per account

## Email Feed Page           
- Filter by category:        (Interview / Offer / Assessment / Rejected)
- Each email card shows:
    - Sender
    - Subject
    - AI Category badge
    - Time received
    - "View Summary" button

## AI Summary Page
- Email Subject
- Sender + Date
- AI Category badge          
- Summary
- Deadline / Action Required
- "Add to Calendar" button   
- "Mark as Read" button      

## Notifications Page        
- List of all alerts
- Unread shown in bold
- "Mark all as read" button

## Settings Page             
- Change Name / Password
- Connected Gmail accounts
- Notification preferences
