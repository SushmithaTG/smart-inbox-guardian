# User Stories

## Story 1

As a student,

I want Smart Inbox Guardian to notify me when I receive an interview email,

so that I never miss an opportunity.

---

## Story 2

As a user,

I want AI to summarize long emails,

so I can quickly understand them.

---

## Story 3

As a job seeker,

I want reminders before interview deadlines,

so I can prepare on time.
